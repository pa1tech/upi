## How to use !ncredible UPI Pay
* To send a specific amount to a particular UPI id, use `https://pa1tech.github.io/upi?<upi_id>&<amt>`
* To send without any specific amount, `https://pa1tech.github.io/upi/index.html?<upi_id>`
